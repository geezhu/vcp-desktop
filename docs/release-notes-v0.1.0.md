# VCPInstallerGUI Release Notes v0.1.0

**Release Date**: 2026-02-24  
**Release Owner**: openclaw

## Summary

This release is a documentation baseline release for the Linux-first installer direction.
No production installer binary is published in this version.

## Included in This Release

1. Initial requirement scope and acceptance criteria.
2. Linux AIO design baseline (AppImage-first distribution strategy and fallback path).
3. Release checklist and task execution convention draft.

## Breaking Changes

None.

## Known Limitations

1. No executable installer implementation in repository yet.
2. AppImage build pipeline is not implemented yet.
3. Published `tar.gz` is a documentation baseline package, not a runnable installer binary.

## Next Milestone

1. Implement minimal installer core for Linux baseline.
2. Add AppImage packaging and artifact signing for executable releases.
